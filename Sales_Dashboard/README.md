# Sales Performance Dashboard
## Objective
Analyze sales data to identify trends, top-performing products, and category-wise performance.
## Tools Used
- Power BI
- Excel
## Key Insights
- Technology category generates highest revenue
- Sales show consistent growth over time
- Top products contribute significant portion of revenue
